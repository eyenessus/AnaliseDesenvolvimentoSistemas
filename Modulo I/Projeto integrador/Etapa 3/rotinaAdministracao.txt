E necessário utilizar software Ottomatik.io, configurar para que realize o backup todas 00h de cada dia, assim sendo a cada 24h apartir do inicio do primeiro backup do banco de dados, os backups devem ser realizado na nuvem como Google Drive, OneDrive assim que o backup for realizado com sucesso ou caso de falha será enviado um e-mail informando hórario e data da ocorrência. Caso de desastres deve realiza a restauração dos dados feito pelo ultimo backup com status de sucesso completo, assim fazendo validações da integridade de todos dados restaurados.
